使用说明

1. 输入需要的材料数量；

2. 点击“过滤”掉与合成无关的材料；

3. 输入已有的材料数量；

4. 再次“过滤”掉不需要刷的材料。

Usage

1. Input the needed number of materials.

2. "Filter" out irrelevant materials.

3. Input material numbers you have.

4. "Filter" out materials you don't need to work for.
